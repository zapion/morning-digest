Datasette Agent

Source: Simon Willison

Published: 2026-05-21T19:52:19+00:00

🔗 https://simonwillison.net/2026/May/21/datasette-agent/#atom-everything

Tags: #ai #news #obsidian #rss #commentary #workflow #llm #multimodal

ID: 1a8a76c365

Summary

我們剛剛宣布推出 Datasette Agent 的第一個版本，這是一個可擴充的全新 Datasette AI 助理。我使用 LLM Python 函式庫已經超過三年，而 Datasette Agent 代表 LLM 和 Datasette 終於真正結合的時刻。我對此非常興奮！Datasette Agent 提供對話式介面，讓你可以針對儲存在 Datasette 中的資料提問。加入 datasette-agent-charts 外掛後，它也能為你的資料生成圖表。示範：公告文章（在新的 Datasette 專案部落格上）包含這段示範影片。我是在新的 agent.datasette.io 線上示範實例上錄製這支影片，該實例使用 Datasette Agent 連接多個範例資料庫，包括 WRI 的經典 global-power-plants，以及我部落格的 Datasette 備份副本。這個線上示範執行在 Gemini 3.1 Flash-Lite 上——它便宜、快速，且很擅長撰寫 SQLite 查詢。我在示範中問的問題是：Simon 最近一次看到鵜鶘是什麼時候？它執行了這段 SQL 查詢：SELECT title, commentary, created FROM blog_beat WHERE beat_type = ' sighting ' AND (title LIKE ' %pelican% ' OR commentary LIKE ' %pelican% ' ) ORDER BY created DESC LIMIT 5 並回答：Simon 最近一次記錄到鵜鶘的目擊是在 2026 年 5 月 20 日。該觀察包含一隻加州褐鵜鶘，以及一隻普通潛鳥、一隻加拿大雁、一隻條紋岸蟹和一隻加州海獅。這是我部落格上的那次目擊，以及完整對話逐字稿的 Markdown 匯出。外掛：Datasette Agent 我最喜歡的功能是，和 Datasette 的其他部分一樣，它可以透過外掛擴充。我們目前已推出三個外掛：datasette-agent-charts（影片中有展示）利用 Observable Plot 為 Datasette Agent 新增圖表功能。datasette-agent-openai-imagegen 使用 ChatGPT Images 2.0，為 Datasette Agent 加入影像生成工具。datasette-agent-sprites 提供在 Fly Sprites 持久化沙箱中執行程式碼的工具。撰寫外掛真的很有趣。我還有很多原型，尚未達到 alpha 品質。Claude Code 和 OpenAI Codex 在撰寫外掛方面都表現出色——只要把它們指向 datasette-agent repo 的檢出版本作為參考，然後告訴它們你想做什麼就行！在本地模型上執行：我也很享受讓這個新外掛在本地模型上運行。以下是一行 uv 指令，可在 Mac 上的 LM Studio 中，針對 gemma-4-26b-a4b 執行該外掛：uvx --prerelease=allow \ --with datasette-agent --with llm-lmstudio \ datasette --internal internal.db --root \ -s plugins.datasette-llm.default_model lmstudio/google/gemma-4-26b-a4b \ data.db Datasette Agent 需要可靠的工具呼叫，以及讓模型產生可在 SQLite 上執行的 SQL 查詢的能力。過去六個月發布的開放權重模型，愈來愈能處理這件事。接下來呢：Datasette Agent 為 LLM 與 Datasette 生態系，乃至更廣泛領域，開啟了許多可能性。它已經影響了主要的 LLM 0.32a0 重構；我幾乎已準備好把它納入穩定版，或許還會從 Datasette Agent 本身抽取出一些額外的「LLM agent」抽象層。我也在探索自己對 Claude Artifacts 的詮釋，作為一個外掛，進展相當不錯。我很期待用 Datasette Agent 建立自己的 Claw——一個以從數位生活不同部分匯入的資料為核心的個人 AI 助理，這也讓我有個很棒的理由重新檢視我早期的 Dogsheep 工具家族。我們也會為 Datasette Cloud 的使用者推出 Datasette Agent。如果你想聊這個專案，歡迎加入我們的 #datasette-agent Discord 頻道。標籤：projects、sqlite、ai、datasette、generative-ai、llms、llm、uv、datasette-agent
